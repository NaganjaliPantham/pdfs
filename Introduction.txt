STAR for Project

During my academics, I worked on a project named Dark web classification. This project mainly aims to classify the text that is present in dark web .The dark web contains a large amount of unstructured and often sensitive textual data, making it challenging to identify meaningful patterns or categorize content effectively. My objective was to build a model that could accurately classify dark web text into relevant categories by leveraging advanced deep learning techniques. Additionally, I aimed to improve classification performance by combining deep learning with topic modeling approaches. I implemented a TextCNN model for classifying dark web text and used topic modeling (LDA) to extract semantic insights. I combined topic distributions with TextCNN features through a topic-weighting approach to improve contextual understanding and overall classification accuracy. The hybrid approach improved classification accuracy and contextual understanding of dark web text while strengthening my skills in deep learning.
